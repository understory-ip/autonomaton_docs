# Autonomaton Pattern Documentation

**About This Repository**

This repository contains the official documentation, architectural specifications, and reference implementation guides for the **Autonomaton Pattern**. 

The Autonomaton Pattern is an open protocol (CC BY 4.0) designed to shift AI capability from centralized, rented infrastructure to sovereign, composable nodes. These docs cover the core invariants of the architecture, including:

* **The 5-Stage Pipeline:** The "hourglass invariant" of cognitive routing.
* **The Cognitive Router & Zone Governance:** Declarative, edge-local control over autonomous execution.
* **The Reverse Tax:** The economic mechanics of downward model migration and structural cost reduction. 
* **Composition Primitives:** How independent Autonomatons scale and federate via telemetry exhaust. 

For the complete Python reference implementation, see the [`grove-autonomaton-primitive` repository](https://github.com/understory-llc/grove-autonomaton-primitive).
